# miruns-website

Marketing website for Miruns, built with Next.js 16, React 19, Tailwind CSS, and Framer Motion.

<img width="1623" height="1197" alt="image" src="https://github.com/user-attachments/assets/a581a5f9-95e6-487f-a981-33f4865e92e0" />

## Dev

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Scripts

| Command | Description |
|---|---|
| `npm run dev` | Start dev server |
| `npm run build` | Production build |
| `npm run start` | Start production server |
| `npm run lint` | Lint |

## Stack

- **Next.js 16** — framework
- **React 19** — UI
- **Tailwind CSS 4** — styling
- **Framer Motion** — animations
